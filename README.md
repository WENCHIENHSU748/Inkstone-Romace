以 Vtuber「硯颺」為女主角做一款戀愛遊戲非常適合，尤其是她帶點懷舊、內斂又神秘的氣質，非常容易吸引多樣化的追求者。以下是五位可攻略角色設定，性別與風格各異，讓玩家可以從多元選項中選擇「誰最懂硯颺」：

🌕 1. 【墨離｜Mo-li】
性別：男

風格：古風×優雅×神祕系

身分背景：國樂系講師兼樂器修復師，自稱「以聲為墨，與君共書」。

個性特徵：內斂穩重，言詞充滿古典詩意，對硯頗有「前世之緣」執念。

追求方式：常寄送詩句、手工墨條，認為硯是他生命中缺失的「另一半靈魂」。

玩家選擇困難點：難以辨別他是真情，還是沉溺於過去幻想。

🔥 2. 【Rika】
性別：女

風格：辣妹系×直播圈女神×直球派

身分背景：人氣美妝實況主，與硯曾有短暫合作，對硯顏控程度極高。

個性特徵：開朗、自信、直來直往，會主動開黃腔但其實心很軟。

追求方式：公開在直播中示愛，私下傳性感自拍與貼心早安問候。

玩家選擇困難點：外表輕浮，實則真心，但硯需判斷是否能接受她的高調與節奏。

☕ 3. 【周一】
性別：女（非二元性別傾向）

風格：文藝×性冷淡×日常哲學家

身分背景：咖啡店老闆，白天營業，晚上寫詩。無社群帳號，僅靠口碑經營。

個性特徵：慢熱寡言，喜歡討論夢、語言與時間的意義，曾說過「你像夜裡的墨水，會染進我心裡」。

玩家選擇困難點：難以確認對方是否真有情感意圖，還是單純習慣與人保持詩意距離。

📱 4. 【星航｜Star】
性別：男

風格：科技宅×天才駭客×輕度病嬌

身分背景：地下網路世界中的知名 AI 模型調教師，懷疑硯是古代遺留的智能生命。

個性特徵：偏執但聰明，對於硯的出身有極強好奇與佔有欲，認為「只有他能解構她」。

玩家選擇困難點：魅力與危險並存，信任與恐懼之間搖擺。

🦊 5. 【阿狐】
性別：不明（可由玩家自行想像，狐性存在）

風格：妖異×可愛×愛吃糖

身分背景：據說是硯台的守靈獸，因硯未下葬而轉世為人形，默默守護她多年。

玩家選擇困難點：選擇牠是否代表與常人世界斷絕、步入超現實的歸宿。
